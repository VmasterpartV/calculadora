Instrucciones

Para realizar una operación matemática, presione los botones de la calculadora y luego presione el botón de igual.

Para borrar el último dígito presione el botón "Del".

Para borrar la operación presione el botón "C".

Para realizar una operación de raíz cuadrada presione el botón "√".

Para realizar una operación de potencia presione el botón "^".

Para realizar una operación de logaritmo presione el botón "log".
Si desea realizar un logaritmo con base específica, deberá ingresar la base y luego el número de la siguiente forma: "log(base, número)".
Si no indica la base, se asumirá que el logaritmo es de base 10, "log(número)".

Para realizar una operación de paréntesis presione los botones "(" y ")".
